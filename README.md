# Call-Center-Performance-Dashboard-Excel-

## Description:
This project presents an interactive Call Center Performance Dashboard built using Microsoft Excel. It provides deep insights into agent productivity, customer satisfaction, and operational efficiency through dynamic visualizations and KPIs.

## Key Features:
📌 KPI Cards: Total Calls, Avg Speed of Answer, Avg Talk Time, Satisfaction Score, Answer Rate, Resolution Rate

📊 Agent-wise Performance Analysis

📈 Hourly Call Volume (Peak Analysis)

📅 Monthly Trend Analysis

🧩 Topic-wise Call Distribution & Resolution

🎯 Answered vs Unanswered Call Breakdown

🔍 Interactive Slicers (Agent, Topic, Month, Answered Status)

## Tools & Techniques Used:
Microsoft Excel

Pivot Tables & Pivot Charts

Slicers for interactivity

Data Cleaning & Transformation

Conditional Formatting

## Outcome:
Identified peak call hours and top-performing agents
Improved visibility into resolution efficiency and customer satisfaction trends
Built a user-friendly dashboard for quick decision-making
